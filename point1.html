포인트 적립/사용

제휴 브랜드 이용 금액에 따라 CJ ONE 포인트를 적립 또는 현금처럼 사용 할 수 있습니다.

  • 적립
  • 사용
  • 01

    적립된 포인트는 가용화를 거쳐 사용 가능한 포인트로 전환됩니다.

  • 02

    온라인 쇼핑몰에서 포인트 받기로 다운받은 포인트는 즉시 가용포인트로 전환됩니다.

  • 03

    결제 시 포인트를 사용할 경우, 포인트 사용 금액을 제외한 지불 금액에 대해 포인트가 적립됩니다.

  • 04

    각 제휴 브랜드별로 포인트 적립/사용이 제외되는 일부 매장 및 상품이 있을 수 있습니다.

  • 05

    각 제휴 브랜드별로 1일 적립 횟수 제한이 있으니 브랜드별 제한 내용을 확인하시기 바랍니다.

  • TIP

    제휴 브랜드 이용실적에 따라 최대 100%까지 추가 포인트를 받으실 수 있습니다.

  • 01

    보유 가용 포인트 1,000포인트 이상일 때 현금처럼 사용 가능합니다.

  • 02

    보유 가용 포인트 10,000 포인트 이상 사용 시, 카드 비밀번호 또는 본인인증이 필요합니다.

  • 03

    CJ ONE 카드 혹은 CJ ONE 모바일 카드 중 최소 1개 이상 카드 등록이 되어 있어야 사용이 가능합니다.

  • 04

    오프라인 매장에서는 카드를 소지한 경우에 한해서만 포인트 사용이 가능합니다. (마그네틱, 모바일/App카드)

  • 05

    각 제휴 브랜드별로 포인트 사용 단위는 상이하며, 자세한 내용은 제휴 브랜드별 유의사항을 확인하세요.

  • TIP

    적립된 포인트의 유효기간과 소멸일을 꼭 확인하세요.

포인트 이용방법

  • 오프라인 매장
    • 결제 시 CJ ONE 카드 제시와 함께 포인트 적립 혹은 사용을 요청하세요.
    • 카드 미 휴대시, 회원님의 휴대전화번호 확인으로 적립 가능합니다.
    • 카드를 소지한 경우에 한해서만 포인트 사용이 가능합니다. (마그네틱, 모바일/App카드)
    • 일부 제휴사의 경우 포인트 사용 시, 카드 비밀번호 입력 기준이 다를 수 있습니다.
  • 온라인 매장
    • CJ ONE 제휴 브랜드 웹사이트의 상품 결제화면에서 잔여 가용포인트를 확인하고 바로 사용할 수 있습니다.
    • CJ ONE 제휴 브랜드 웹사이트에서 로그인 후 구매 혹은 결제하시면 자동 적립됩니다.
  • 유효기간 체크
    • 포인트 유효기간
      • 기본 적립 포인트 & 보너스포인트 : 24개월 (일반 상품 / 서비스 구매 시 적립)
      • 선물 받은 포인트 : 6개월 (선물 받은 날 부터!)
      • 카드를 소지한 경우에 한해서만 포인트 사용이 가능합니다. (마그네틱, 모바일/App카드)
      • 이벤트로 받은 포인트 : 이벤트마다 유효기간 다르게 지정 (이벤트 당첨 시 확인 필수!)
    • 포인트 소멸
      • 소멸예정포인트는 유효기간 종료일이 다가와 소멸이 예상되는 포인트입니다.
      • 유효기간이 경과된 포인트는 유효기간 종료되는 해당 월 말일에 소멸됩니다.
      • 소멸 기준은 잔여 가용포인트 중 유효기간이 가장 짧은 포인트부터 소멸되며, 소멸된 포인트는 복구되지 않습니다.
      • 포인트 사용취소 시점에 해당 포인트의 유효기간이 경과한 경우, 당일까지만 사용 가능하며 소멸된 이후 복구되지 않습니다.
